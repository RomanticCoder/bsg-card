# React E-commerce web application
- Collaborated with a graphic designer and created a modern application.
- Utilized Firebase RESTful APIs and integrated them with front-end code using Next.js.
- Implemented OAuth2 authentication with Firebase.
- Fixed bugs and implemented new features required for launching the product.
